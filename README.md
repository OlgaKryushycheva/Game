# Поле чудес (Java)

Консольна реалізація гри "Поле чудес" з режимом проти ШІ та мультиплеєром (2–4 гравці). Проєкт побудований на Gradle і використовує Java 21 (перевірено також із JDK 25).

## Запуск

1. Переконайтеся, що у вас встановлений JDK 17+ (перевірено з JDK 25) і використовується в якості Gradle JVM.
2. Якщо файлу `gradle/wrapper/gradle-wrapper.jar` немає (бінарні файли не зберігаються у репозиторії), згенеруйте його локально:

   ```bash
   gradle wrapper --gradle-version 8.17
   ```

3. Якщо прямий доступ до `https://services.gradle.org/distributions/gradle-8.17-bin.zip` недоступний, завантажте цей ZIP із машини з інтернетом і покладіть у кеш `~/.gradle/wrapper/dists/` (повну структуру шляху Gradle створить автоматично після першої спроби запуску wrapper). Це прибирає спроби звернення до GitHub-дзеркал під час старту.

4. Запустіть гру:

   ```bash
   ./gradlew :app:run
   ```

> Примітка: обгортка Gradle налаштована на версію 8.17, що підтримує запуск із JDK 25. Якщо потрібно перестворити wrapper локально, використовуйте цю ж або новішу версію Gradle, щоб уникнути несумісності JVM.

## Можливості
- Режим проти ШІ (ШІ обирає букви за частотністю та може вгадати слово за шаблоном).
- Режим для декількох гравців.
- Обертання барабану з сектором "Банкрут" та "Пропуск".
- Таблиця результатів зберігається у файлі `scoreboard.txt`.

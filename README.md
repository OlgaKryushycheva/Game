# Поле чудес (Java)

Консольна реалізація гри "Поле чудес" з режимом проти ШІ та мультиплеєром (2–4 гравці). Проєкт побудований на Gradle і використовує Java 21.

## Запуск

1. Переконайтеся, що у вас встановлений локальний Gradle (8.14.3 або сумісний).
2. Згенеруйте відсутній wrapper JAR (бінарні файли не додаються до репозиторію):

   ```bash
   gradle wrapper --gradle-version 8.14.3
   ```

3. Запустіть гру:

   ```bash
   ./gradlew :app:run
   ```

## Можливості
- Режим проти ШІ (ШІ обирає букви за частотністю та може вгадати слово за шаблоном).
- Режим для декількох гравців.
- Обертання барабану з сектором "Банкрут" та "Пропуск".
- Таблиця результатів зберігається у файлі `scoreboard.txt`.

# Поле чудес (Java)

Консольна реалізація гри "Поле чудес" з режимом проти ШІ та мультиплеєром (2–4 гравці). Проєкт побудований на Gradle і використовує JDK 25.

## Запуск

### Якщо у вас уже є JDK 25
1. Встановіть **локальний Gradle 9.2.1** (потрібен лише для генерації відсутнього `gradle/wrapper/gradle-wrapper.jar`). Беріть архів із каталогу дистрибутивів, а не з GitHub mirror:
   - Завантажте ZIP 9.2.1 із каталогу https://services.gradle.org/distributions/ (файл `gradle-9.2.1-bin.zip`).
   - Розпакуйте архів у зручну теку, наприклад `C:\Gradle\gradle-9.2.1` або `/opt/gradle/gradle-9.2.1`.
   - Додайте шлях до `bin` у `PATH` (Windows: *Система → Додаткові параметри → Змінні середовища*, змінна `Path`; Linux/macOS: додайте `export PATH=/opt/gradle/gradle-9.2.1/bin:$PATH` до свого шеллу).
   - Перевірте в терміналі IntelliJ командою `gradle -v` — має показати версію 9.2.1.
2. Згенеруйте wrapper, якщо `gradle/wrapper/gradle-wrapper.jar` відсутній:

   ```bash
   gradle wrapper --gradle-version 9.2.1
   ```

   Після цього замість глобального Gradle можна користуватися лише `./gradlew`.
3. Якщо прямий доступ до `https://services.gradle.org/distributions/gradle-9.2.1-bin.zip` недоступний:
   - Завантажте ZIP на машині з інтернетом і перенесіть файл.
   - Запустіть `./gradlew --version` один раз, щоб він створив структуру кешу в `~/.gradle/wrapper/dists/`.
   - Скопіюйте `gradle-9.2.1-bin.zip` у створену підпапку `~/.gradle/wrapper/dists/gradle-9.2.1-bin/<hash>/`. Назву `<hash>` можна піддивитися у створеній директорії.
   - Після цього wrapper не буде нічого качати.

### Як запустити гру (консоль)
1. **Через Gradle wrapper (рекомендовано):**

   ```bash
   ./gradlew :app:run
   ```

   - Windows: у вбудованому терміналі IntelliJ введіть `.\gradlew.bat :app:run`.
   - "Звичайний термінал" = будь-яке вікно, де ви реально можете набирати текст (PowerShell, CMD, Terminal на macOS/Linux). Не плутайте з вкладкою *Gradle* у IDE, яка часто не передає інтерактивний ввід.
   - Якщо Gradle запущено без можливості вводити текст, гра побачить відсутність вводу й завершиться з підказкою відкрити звичайний термінал.
2. **Через IntelliJ IDEA (якщо хочете кнопкою Run):**
   - Відкрийте проєкт **File → Open…** і виберіть корінь репозиторію; IntelliJ сам підхопить Gradle-проєкт.
   - У верхньому правому куті натисніть **Add Configuration…** → **+** → **Application**.
   - Поле *Name*: наприклад, `PoleChudes`.
   - *Main class*: `game.App` (натисніть `...` і виберіть його, або впишіть вручну).
   - *Working directory*: шлях до кореня проєкту (там, де лежать `settings.gradle` та `app/`).
   - Натисніть **OK**, далі кнопку **Run** (зелена стрілка). Уся взаємодія з грою буде у консолі Run внизу IDE; вводьте числа/літери безпосередньо в це вікно.

Після старту ви побачите текстове меню з вибором режиму (гра проти ШІ або мультиплеєр), ходом по гравцях і барабаном. Результати зберігаються у файлі `scoreboard.txt` у корені проєкту (створюється автоматично після першої гри).

## Можливості
- Режим проти ШІ (ШІ обирає букви за частотністю та може вгадати слово за шаблоном).
- Режим для декількох гравців.
- Обертання барабану з сектором "Банкрут" та "Пропуск".
- Таблиця результатів зберігається у файлі `scoreboard.txt`.
